README.md

A m×nm\times nm×n board yields a system of linear equations in mnmnmn variables. Each variable aija_{ij}aij​ represents the number of mines at row iii, column jjj. Each equation sets a constraint on how many mines can be at a location’s neighbors based on the number in its tile. For example, if there are b23b_{23}b23​ mines adjacent to a non-edge tile at row 222, column 333, the correpsonding equation is a13+a22+a33+a24=b23a_{13} + a_{22} + a_{33} + a_{24} = b_{23}a13​+a22​+a33​+a24​=b23​.

Normally, such systems are solved using Gaussian elimination in O((mn)3)O((mn)^3)O((mn)3). Luckily, our system of equations is very sparse, so scipy.sparse can solve the system more quickly.
